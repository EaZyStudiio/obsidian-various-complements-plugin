## 🖥️ For developers

- Requirements
    - Node.js v22

### Development

#### Set up

```bash
git config core.hooksPath hooks
```

#### Install dependencies

```bash
corepack enable
pnpm install
```

#### Build for development

```bash
pnpm dev
```

#### Test

```bash
pnpm test
# or
pnpm test --watch
```

#### CI

```bash
pnpm run ci
```

#### Release

```bash
# Stable
VERSION=1.2.3 pnpm release

# Beta
VERSION=1.2.3-beta1 pnpm release
```
